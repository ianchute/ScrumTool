### Your Open Source Scrum/Kanban Process Management Tool

#### By: Ian Chu Te

#### Setup:

To setup: `npm install && sudo npm -g install react-scripts`

To run: `npm start`

To build production bundle: `react-scripts build`

#### Stack:

  * React
  * Redux
  * ImmutableJS
  * SemanticUI (Stardust)
  * React-Scripts (a.k.a. create-react-app)
  * UnderscoreJS
